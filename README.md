# 📊 Retail Sales Forecasting Dashboard

## 🔍 Overview
This project analyzes 4 years of retail sales data (2014–2017) to generate business insights and forecast future sales.

## 🎯 Business Problem
Retail businesses struggle to:
- Identify top-performing products
- Understand discount impact on profit
- Forecast future sales trends

## 🛠 Tools Used
- MySQL Workbench
- Power BI
- SQL
- DAX

## 📈 Key Insights
- 33% of orders are loss-making due to heavy discounts
- Furniture is the top revenue category
- Consumer segment dominates sales
- Discounts above 20% reduce profitability

## 🔮 Forecast
12-month forecast using Exponential Smoothing (ETS)
